## API

| 属性    | 说明                         | 类型         | 默认值  |
| :------ | :--------------------------- | :----------- | :------ |
| visible | 控制显示/隐藏                | `boolean`    | `false` |
| urls    | 图片文件数组                 | `string[]`   | `[]`    |
| onClose | 关闭回调                     | `() => void` | -       |
| index   | 初始化展示图片在数组中的下标 | `number`     | `0`     |
